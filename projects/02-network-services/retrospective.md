# Retrospective

## What I learned
- How to verify whether SSH is running on an Ubuntu server
- How to install, start, and enable Nginx
- How to confirm services are listening on expected ports
- How to test a service locally on the VM and remotely from another machine
- How to check whether the Ubuntu firewall is active

## What was confusing
- I mixed up commands that belong on the Mac versus commands that belong on the Ubuntu VM
- I initially started the terminal recording from the wrong directory

## What I would do better next time
- Start evidence capture only after confirming I am in the correct lab folder
- Keep one terminal for the Mac and one for the SSH session so I do not mix environments
- Record the lab objective at the start before doing any commands
