# Risk Scoring

## Low Risk
- Vendor accesses only low-sensitivity data
- SSO is supported
- MFA is required
- Data is encrypted at rest and in transit
- Security documentation is available

## Medium Risk
- Vendor accesses internal business data
- One or two security controls are missing
- Security documentation is incomplete

## High Risk
- Vendor accesses sensitive customer or regulated data
- MFA is not required
- Encryption is missing
- No security documentation is available
- Data is stored in higher-risk ways without clear controls
