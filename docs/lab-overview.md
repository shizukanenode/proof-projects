# Lab Overview

## Purpose
This lab is designed to build practical IT and cybersecurity skills through small, repeatable, documented projects.

## Host Environment
- Host machine: MacBook
- Repository location: `~/Desktop/Projects/proof-projects`
- Workflow: Terminal, Git, Markdown, Python, and virtual machines

## Lab Design Principles
- Keep projects small and real
- Document what broke and how it was fixed
- Prefer repeatable exercises over random tool collection
- Keep public evidence clean and sanitized
- Focus on employable skills, not hype projects

## Primary Skill Areas
1. Linux administration
2. Networking and services
3. Log analysis and triage
4. Backup and recovery
5. Automation with Bash and Python

## Evidence Standard
Each project should include:
- Objective
- Environment
- Files used
- Steps performed
- Problems encountered
- Resolution
- Skills demonstrated
- Future improvements
