# Project Summaries

## IAM Device Tracker
Built a lightweight IAM-style workflow project using CSV data, onboarding/offboarding scripts, and audit logging. Demonstrates basic identity lifecycle handling, least privilege thinking, and simple Python automation.

## SOC Triage Lab
Built a small SOC-style triage project with sample logs, a parser, detections, and an incident report. Demonstrates log analysis, event triage, basic detection thinking, and security documentation.

## Vendor Intake App
Built a simple vendor intake and review application with questionnaire and risk scoring support. Demonstrates security review workflow thinking, lightweight application structure, and risk documentation.

## Linux Admin Basics
Planned lab project to demonstrate practical Linux administration, service checks, permissions, system health validation, and troubleshooting.

## Network Services
Planned lab project to demonstrate service configuration, connectivity testing, and basic network troubleshooting.

## Log Analysis
Planned lab project to expand detection, event review, and incident handling skills.

## Backup and Recovery
Planned lab project to demonstrate operational resilience and restore validation.

## Python Automation
Planned lab project focused on utility scripts for health checks, reporting, and repeatable administrative tasks.
